## Consideraciones

* Datos estan ordenados a traves del tiempo.
* Id de Usuarios Unicos.
* Formato de Intervalo de Fecha: AAAA-MM-DD_AAAA-MM-DD
* Implementado Carpeta functions y data en src.
* En data.service la variable Data almacena la ruta del archivo.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
